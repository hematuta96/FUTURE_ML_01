# 📈 AI-Powered Sales & Demand Forecasting System

## 🚀 Future Interns - Machine Learning Task 1

This project predicts future sales trends using Machine Learning techniques and historical sales data. The goal is to help businesses make data-driven decisions for inventory management, demand planning, and sales forecasting.

---

## 📌 Project Overview

The system analyzes historical sales records from a Superstore dataset and forecasts future sales using a Linear Regression model.

### Objectives
- Analyze historical sales trends
- Build a machine learning forecasting model
- Predict future sales
- Visualize sales patterns and forecasts

---

Dataset Used: Superstore Sales Dataset

Dataset Features:
- Order Date

- Product Information
- Category
- Region
- Sales

---

## ⚙️ Machine Learning Workflow

### 1. Data Collection
Loaded the Superstore sales dataset.

### 2. Data Preprocessing
- Converted date columns into datetime format
- Aggregated sales by order date
- Created numerical features for model training

### 3. Model Building
Used Linear Regression to learn sales patterns from historical data.

### 4. Prediction
Forecasted future sales for upcoming days.

### 5. Visualization
Generated graphs to visualize:
- Historical Sales Trends
- Future Sales Forecasts
